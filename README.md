Game backlog manager:

CHECKPOINT
------------------------------

This project is a Java application that lets users manage a video game backlog using a simple GUI built with Java Swing and an SQLite database. Users can add, update, delete, search, and filter games, as well as view quick statistics about their backlog.

== Features ==
------------------------------

Connect to any SQLite database file

Add / update / delete game entries

Search and filter by name, platform, status, or ownership

Status chips and counters (Unplayed, Playing, Beaten, Total)

Import games from a formatted text file

Generate a backlog report

Fully documented with Javadoc

== Technologies Used ==
------------------------------

Java 22

Java Swing

SQLite (via JDBC)

IntelliJ IDEA

java doc

== How to Run == 
------------------------------
Open the project in IntelliJ.

Make sure the SQLite JDBC driver JAR is added as a project library.

Run CheckPointSwing.java.

Select or create a database file when prompted.

CheckPoint Game Tracker

Author: Stephanie Ortiz
Course: CEN 3024 - Software Development 1
Date: October 2025

How to Run:
1. Make sure Java 17 or newer is installed.
   (Check by typing "java -version" in Command Prompt)

2. Keep both CheckPoint.jar and games.txt in the same folder.

3. Double-click CheckPoint.jar
   OR open Command Prompt here and type:
   java -jar CheckPoint.jar
